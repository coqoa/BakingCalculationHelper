## Planning

### 컨셉
- 베이킹 재료를 계량하는데 도움이 되는 계산 앱

### 필수 기능
- 회원가입 / 로그인
- 레시피 CRUD
- 레시피 배합 계산기

### 부가 기능
- 웹 / 앱(앱스토어, 플레이스토어) / 크롬확장프로그램 플랫폼 이용
- 레시피 정렬
- 다중 기기 동시접속
- 한글 / 영어 제공
- 스크린 이동, 페이지 이동
- 광고

`디자인, 기획은 추후 변경이 가능`
